# Introduction

This page contains, based on the [Arc42](https://arc42.org/more), the architectural documentation of the MySite360.

## Document Structure

This document claims to describe the high level architecture of the project and concentrates on a holistic system wide overview.

This documentation is maintained and hosted on [code.siemens.com](https://code.siemens.com/mydevops/mysite360/tasks_and_docs) and continously updated. The up to date state can be found [as gitlab pages](https://tasks-and-docs-119b90.code.siemens.io/).

Detailed designs are described in distributed code repositories and are referenced via hyperlinks from this document.
