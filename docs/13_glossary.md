# Glossary

**Contents**

The most important domain and technical terms that your stakeholders use
when discussing the system.

**Abreviations**

<!-- markdownlint-disable MD041 -->

|Abreviation|Long description|Comment/Reference|
|:---|:---|:---|
|_IAH_|Industrial Asset Hub|Overall term for the product offering containing multiple deployments (SaaS, SDK, Edge Apps, Compose)|
|_CDM_|Common Device Management|Internal project name|
|_KDD_|Key Design Decision|Significant architectural decision. Elsewhere often named as ASR (Architecture Significant Requirement) or ADR (Architecture Design Record)|
|_WFX_|Workflow Executor| Modelplayer allowing to execute statemachines with distributed locked state from Edge to Cloud|
|_XO_|Xcelerator Operations|The executive part of SIEMENS Xcelerator|
|_XF_|Xccelerator Foundation|Technical foundational services around deployment, authentication, entitlement operation etc.. Sometimes as well referred as _FDS_ (Foundational Services)|
|_OT_|Operational Technology|OT defines operational technologies for industries connected production setups|
|_API_|Application Programming Interface||
|_COTS_|Commercial off the shelf|Commercial software|
|_OSS_|Open Source Software||
